# RunProject

## 📖 项目简介

RunProject 是一个基于 Tauri + React + Rust 构建的高效开发者工作台。它旨在统一管理多个 Node.js 项目，提供流畅的脚本执行、环境切换（NVM）及终端管理体验，帮助开发者从繁琐的命令行操作中解放出来。

## ✨ 当前核心特性

- **工作区管理**：一键扫描目录，自动识别 npm/yarn/pnpm 项目。
- **智能脚本解析**：自动解析 `package.json` scripts，可视化点击运行。
- **环境隔离**：深度集成 NVM，支持为不同项目自动切换 Node.js 版本。
- **Kitty 终端集成**：利用高性能 Kitty 终端执行任务，支持多标签页管理。
- **数据格式化**：支持 JSON、XML、SQL、CSS、HTML、JavaScript 等多种格式的美化和压缩。
- **现代化 UI**：基于 Shadcn/UI 设计，简洁美观，支持亮/暗模式。
- **Git 基础集成**：查看和切换分支。

## 🚀 发展规划与走向 (Roadmap)

### 阶段一：体验打磨与基础增强 (短期)

- [*] **终端体验优化**：增强与 Kitty 的通信稳定性，增加内嵌终端 (xterm.js) 作为备选方案，适配更多终端模拟器。
- [ ] **任务日志持久化**：保存任务运行历史和日志，便于排查错误。
- [ ] **快捷键支持**：全键盘操作支持，快速搜索项目和命令 (Command Palette)。
- [ ] **Windows 适配**：解决当前对 Shell 和路径的依赖，实现跨平台支持。

### 阶段二：工作流与能力扩展 (中期)

- [ ] **任务编排**：支持定义“任务组”，一键启动前后端服务、数据库等依赖。
- [ ] **深度 Git 管理**：集成提交、推送、拉取、Diff 查看及简单的冲突解决。
- [ ] **Docker/容器集成**：查看和管理项目关联的 Docker 容器及 Compose 服务。
- [ ] **依赖可视化**：分析项目依赖树，提示版本更新和安全漏洞。

### 阶段三：智能化与生态 (长期)

- [ ] **AI 辅助 (MCP)**：利用 MCP 协议集成 AI 能力，实现智能报错分析、脚本自动生成、代码解释。
- [ ] **插件系统**：开放 API，允许社区开发插件扩展功能（如数据库 GUI、API 调试工具）。
- [ ] **云端同步**：同步工作区配置和常用命令到云端，多设备无缝切换。
