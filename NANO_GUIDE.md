## 一、快速入门

### 1.1 什么是Nano

Nano是一个简单易用的命令行文本编辑器，具有以下特点：

- 界面简洁，操作直观
- 快捷键显示在屏幕底部，随时可见
- 学习曲线平缓，适合初学者
- 支持语法高亮
- 适合快速查看和编辑文本文件

### 1.2 启动Nano

#### 打开新文件

```bash
nano filename.txt
```

#### 打开已存在的文件

```bash
nano /path/to/file.txt
```

#### 打开文件并跳转到指定行

```bash
nano +10 filename.txt
```

这会打开文件并将光标定位到第10行。

#### 打开文件并跳转到指定列

```bash
nano +10,5 filename.txt
```

这会打开文件并将光标定位到第10行第5列。

### 1.3 退出Nano

#### 保存并退出

1. 按 `Ctrl + X`
2. 系统会提示是否保存修改
3. 按 `Y` 确认保存
4. 按回车键确认文件名

#### 不保存直接退出

1. 按 `Ctrl + X`
2. 系统会提示是否保存修改
3. 按 `N` 不保存
4. 直接退出

#### 强制退出（不保存任何修改）

如果文件没有修改过，直接按 `Ctrl + X` 即可退出。

### 1.4 基本光标移动

#### 使用方向键

- `↑` 向上移动一行
- `↓` 向下移动一行
- `←` 向左移动一个字符
- `→` 向右移动一个字符

#### 使用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + P` | 向上移动一行 |
| `Ctrl + N` | 向下移动一行 |
| `Ctrl + B` | 向左移动一个字符 |
| `Ctrl + F` | 向右移动一个字符 |

#### 按词移动

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + ←` | 向左移动一个单词 |
| `Ctrl + →` | 向右移动一个单词 |

#### 按行/段落移动

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + A` | 移动到行首 |
| `Ctrl + E` | 移动到行尾 |
| `Ctrl + V` | 向下翻页 |
| `Ctrl + Y` | 向上翻页 |
| `Home` | 移动到行首 |
| `End` | 移动到行尾 |
| `Page Up` | 向上翻页 |
| `Page Down` | 向下翻页 |

### 1.5 输入和删除文本

#### 输入文本

直接在光标位置输入文字即可，无需进入任何特殊模式。

#### 删除字符

| 快捷键 | 功能 |
|--------|------|
| `Backspace` | 删除光标左侧的字符 |
| `Delete` | 删除光标右侧的字符 |
| `Ctrl + D` | 删除光标右侧的字符 |

#### 删除整行

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + K` | 剪切（删除）当前行 |

## 二、常用操作

### 2.1 查找指定字符

#### 基本查找步骤

1. 按 `Ctrl + W` 打开查找功能
2. 输入要查找的文本
3. 按回车键开始查找
4. 光标会跳转到第一个匹配的位置

#### 查找下一个匹配项

查找完成后，可以继续查找下一个匹配项：

1. 按 `Ctrl + W`
2. 直接按回车键（不输入任何文本）
3. 光标会跳转到下一个匹配位置

#### 反向查找

如果需要从当前位置向上查找：

1. 按 `Ctrl + W`
2. 按 `Ctrl + R` 进入反向查找模式
3. 输入要查找的文本
4. 按回车键开始反向查找

#### 查找示例

假设我们要在文件中查找 "error" 这个词：

```bash
# 步骤1：打开文件
nano log.txt

# 步骤2：按 Ctrl + W

# 步骤3：输入 error

# 步骤4：按回车键

# 步骤5：光标会跳转到第一个 "error" 的位置

# 步骤6：继续查找下一个，按 Ctrl + W，然后按回车键
```

#### 查找技巧

- **大小写敏感**：默认情况下，nano的查找是大小写敏感的
- **查找单词**：如果要查找完整的单词，输入时加上空格
- **查找特殊字符**：可以查找任何字符，包括标点符号

### 2.2 查找并替换

#### 基本替换步骤

1. 按 `Ctrl + \`（反斜杠）
2. 输入要查找的文本
3. 按回车键
4. 输入替换后的文本
5. 按回车键
6. 系统会询问是否替换，按 `Y` 替换，按 `N` 跳过

#### 替换所有匹配项

如果要一次性替换所有匹配项：

1. 按 `Ctrl + \`
2. 输入要查找的文本
3. 按回车键
4. 输入替换后的文本
5. 按回车键
6. 按 `A` 替换所有匹配项

#### 替换示例

假设要将文件中的 "oldname" 替换为 "newname"：

```bash
# 步骤1：打开文件
nano config.txt

# 步骤2：按 Ctrl + \

# 步骤3：输入 oldname

# 步骤4：按回车键

# 步骤5：输入 newname

# 步骤6：按回车键

# 步骤7：按 A 替换所有匹配项
```

### 2.3 复制、剪切、粘贴

#### 剪切（删除）文本

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + K` | 剪切当前行 |
| `Alt + K` | 剪切从光标到行尾的内容 |

连续按 `Ctrl + K` 可以剪切多行，这些行会被存储在剪贴板中。

#### 复制文本

Nano没有直接的复制命令，需要先剪切再粘贴到目标位置。

#### 粘贴文本

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + U` | 粘贴剪贴板中的内容 |

#### 复制多行的方法

如果要复制多行：

1. 移动到要复制的第一行
2. 按 `Ctrl + K` 剪切该行
3. 移动到要复制的最后一行
4. 按 `Ctrl + K` 剪切该行（现在剪贴板中包含了多行）
5. 移动到目标位置
6. 按 `Ctrl + U` 粘贴
7. 移动回原位置
8. 按 `Ctrl + U` 粘贴（恢复原内容）

### 2.4 保存文件

#### 保存当前文件

1. 按 `Ctrl + O`（字母O，不是数字0）
2. 按回车键确认文件名
3. 文件已保存

#### 另存为新文件

1. 按 `Ctrl + O`
2. 修改文件名
3. 按回车键保存

#### 保存并退出

参见 [1.3 退出Nano](#13-退出nano)

## 三、进阶功能

### 3.1 跳转到指定行

#### 方法一：启动时跳转

```bash
nano +100 filename.txt
```

这会打开文件并将光标定位到第100行。

#### 方法二：编辑时跳转

1. 按 `Ctrl + _`（下划线）
2. 输入行号
3. 按回车键

#### 跳转到指定列

1. 按 `Ctrl + _`
2. 输入行号和列号，格式为 `行号,列号`
3. 按回车键

例如，要跳转到第50行第10列：

```bash
# 按 Ctrl + _
# 输入 50,10
# 按回车键
```

### 3.2 显示行号

#### 临时显示行号

1. 按 `Alt + #`（Alt键加井号键）
2. 行号会显示在编辑器左侧
3. 再次按 `Alt + #` 可以隐藏行号

#### 永久显示行号

编辑nano配置文件：

```bash
nano ~/.nanorc
```

添加以下内容：

```
set linenumbers
```

保存后，每次启动nano都会显示行号。

### 3.3 撤销操作

#### 撤销上一步操作

| 快捷键 | 功能 |
|--------|------|
| `Alt + U` | 撤销上一次操作 |

#### 重做操作

| 快捷键 | 功能 |
|--------|------|
| `Alt + E` | 重做被撤销的操作 |

#### 撤销示例

```bash
# 假设你删除了一行
# 按 Alt + U 可以恢复这行

# 假设你输入了一些文字
# 按 Alt + U 可以删除这些文字
```

### 3.4 多文件编辑

#### 打开多个文件

```bash
nano file1.txt file2.txt file3.txt
```

#### 在文件之间切换

| 快捷键 | 功能 |
|--------|------|
| `Alt + <` | 切换到上一个文件 |
| `Alt + >` | 切换到下一个文件 |
| `Alt + .` | 切换到上一个文件（另一种方式） |

#### 在编辑时打开新文件

1. 按 `Ctrl + R`
2. 输入要打开的文件名
3. 按回车键
4. 文件内容会被插入到当前光标位置

### 4.4 实战案例：编写脚本

#### 场景：创建一个简单的Shell脚本

```bash

# 步骤2：输入脚本内容

# 按 Ctrl + O

# 按 Ctrl + X
# 按 Y 确认保存
# 按回车键

# 步骤6：运行脚本
./script.sh
```

```bash
# 移动光标到要修改的位置
# 进行编辑

# 步骤4：查找另一个函数
# 按 Ctrl + W
# 输入 "def process"
# 按回车键

# 步骤5：保存文件
# 按 Ctrl + O
# 按回车键

# 步骤6：退出nano
# 按 Ctrl + X
# 按 Y 确认保存
# 按回车键
```

### 常见问题

#### Q: 如何在nano中复制文本？

A: Nano没有直接的复制命令。方法是先剪切（Ctrl + K），然后粘贴（Ctrl + U）到目标位置，再回到原位置粘贴（Ctrl + U）恢复。

#### Q: 如何取消正在进行的查找？

A: 按 `Ctrl + C` 可以取消查找操作。

#### Q: 如何查看nano的帮助信息？

A: 在nano中按 `Ctrl + G` 可以查看内置的帮助文档。

#### Q: 如何配置nano的默认设置？

A: 编辑 `~/.nanorc` 文件，可以设置行号、语法高亮等选项。

#### Q: 如何在nano中显示特殊字符？

A: 按 `Alt + P` 可以显示或隐藏特殊字符。
